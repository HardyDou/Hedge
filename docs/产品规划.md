# Hedge 产品规划

**产品定位**: Local-First 跨平台密码管理器
**核心价值**: 隐私至上 · 原生体验 · 零成本
**当前版本**: v1.8.0
**更新日期**: 2026-03-13

---

## 📊 当前状态

### ✅ 已完成功能（v1.7.0）

#### 核心功能
- 密码管理（增删改查）
- AES-256-GCM 加密 + Argon2id 密钥派生
- 生物识别解锁（Face ID/Touch ID/指纹）
- 自动锁定
- 附件支持
- **TOTP/2FA 支持**（扫码 + 手动输入，6 位验证码 + 倒计时）

#### 平台支持
- iOS/Android/macOS/Linux/Windows 全平台

#### 同步功能
- 本地存储
- iCloud Drive（iOS/macOS）
- WebDAV（所有平台）
- 冲突检测与备份

#### 用户体验
- 100% Cupertino 设计
- 深色模式
- 拼音排序（数字→字母→拼音）
- 移动端字母索引栏
- 桌面端双栏布局
- Markdown 编辑器
- 实时搜索（支持拼音）

#### 桌面端特色
- 系统托盘常驻
- 快捷面板（悬停显示详情）
- 后台运行

#### 数据导入
- Chrome/1Password/Bitwarden CSV
